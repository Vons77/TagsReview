tags html

  • ol - Utilizada para que possa criar listas que sigam uma sequência clara e ordenada.
  • ul - Utilizada para que possa criar listas sem uma necessidade de ordem clara e bem definidada.
  • a - tag base para gerar links
  • p - definição de parágrafos
  • br salto de linhas
  • img - tag fonte para imagens
  • h1/h2/h... - tags para a definição dos títulos e suas hierarquias
  • css - ferramenta base para definição dos estilos de um site, englobando tudo aquilo que for referente ao design do mesmo
  • abbr - tag utilizada para abreviação de textos
  • code - tag utilizada para formatar um texto no modelo de um código
  • strong - Passar a ideia de força de um trecho ou palavra através do negrito
  • em - Passar a ideia de ênfase através do itálico
  • mark - Realizar a marcação de um trecho
  • pre - realiza a pre formatação de um texto dentro do proprio arquivo .html
  • q - tag para citação
  • blockquote - tag para citação de "blocos" de texto
  • sup - tag para deixar um texto superescrito
  • sub - tag para deixar um texto subescrito
  • dl dt dd - tags para listas de definição
essas e outras tags podem ser facilmente encontradas no site da w3schools.
